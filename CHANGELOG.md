## [0.1.0] - 2026-03-29

### Added

- Initial release of the Laravel CA PKI meta-package.
- Requires all 15 Laravel CA sub-packages (`laravel-ca`, `laravel-ca-key`, `laravel-ca-csr`, `laravel-ca-crt`, `laravel-ca-crl`, `laravel-ca-ocsp`, `laravel-ca-pkcs12`, `laravel-ca-oid`, `laravel-ca-acme`, `laravel-ca-scep`, `laravel-ca-est`, `laravel-ca-tsa`, `laravel-ca-cms`, `laravel-ca-policy`, `laravel-ca-ui`).
- Single `composer require groupesti/laravel-ca-pki` command to install the entire PKI system.
